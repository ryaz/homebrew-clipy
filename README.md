# homebrew-clipy

Homebrew tap for [Clipy](https://github.com/ryaz/clipy-modern) — a modern clipboard manager for macOS.

## Install

```bash
brew tap ryaz/clipy
brew install --cask clipy-modern
```

## After installing

1. Right-click the app > **Open** > **Open** (or run `xattr -cr /Applications/Clipy.app`)
2. Grant **Accessibility** in **System Settings > Privacy & Security > Accessibility**

## Usage

- **Cmd+Shift+V** — Open clipboard history
- **1-9** — Quick-paste item from the menu
